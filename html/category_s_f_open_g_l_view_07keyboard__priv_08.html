Подробное описание

Here are defined a few private messages for keyboard handling in SFOpenGLView.

Документация метода

◆ convertNSKeyEventToSFMLEvent:

+ Event: (NSEvent *)  event

Convert a key down/up NSEvent into an SFML key event.

The conversion is based on localizedKeys and nonLocalizedKeys functions.

Аргументы
eventa key event
Возвращает
sf::Keyboard::Unknown as Code if the key is unknown

◆ isValidTextUnicode:

+ (BOOL) isValidTextUnicode: (NSEvent *)  event

Check if the event represent some Unicode text.

The event is assumed to be a key down event. False is returned if the event is either escape or a non text Unicode.

Аргументы
eventa key down event
Возвращает
true if event represents a Unicode character, false otherwise
